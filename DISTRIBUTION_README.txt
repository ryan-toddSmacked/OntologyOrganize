# ClassifierOrganizer v2.0 - Distribution Package

## What's Included
This is a standalone executable version of ClassifierOrganizer. No Python installation required!

## System Requirements
- Windows 10 or later (64-bit)
- Minimum 4GB RAM recommended
- 500MB free disk space

## Installation
1. Extract all files from this zip archive to a folder of your choice
2. Run `ClassifierOrganizer.exe`

## Features
- Image classification and organization
- Multiple image transformations (edge detection, filters, etc.)
- Advanced sorting by similarity metrics
- Multiple colormap options
- Progress saving and loading
- Export to JSON, CSV, or organized folders

## Quick Start
1. Click "File → Open Folder..." to load images
2. Use the classification panel on the right to create labels
3. Click images to assign labels
4. Use "Transform" menu to apply image filters
5. Use "Sort" menu to organize by similarity
6. Save your progress with "Progress → Save Progress..."
